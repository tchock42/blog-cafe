# Blog de Café

Este es un proyecto de un blog dedicado a la cultura del café. Aquí encontrarás artículos, recetas, y todo lo relacionado con el mundo del café.

## Estructura del Proyecto

- **contacto.html**: Página de contacto.
- **cursos.html**: Información sobre cursos relacionados con el café.
- **entrada.html**: Página para entradas de blog.
- **index.html**: Página principal del blog.
- **nosotros.html**: Información sobre el equipo detrás del blog.
- **css/**: Carpeta que contiene los estilos CSS.
  - `normalize.css`: Archivo para normalizar estilos entre navegadores.
  - `styles.css`: Archivo principal de estilos.
- **img/**: Carpeta que contiene imágenes utilizadas en el blog.
- **js/**: Carpeta que contiene scripts JavaScript.
  - `modernizer.js`: Script para detectar características del navegador.
  - `script.js`: Script principal del blog.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tchock42/blog-cafe.git
   ```
2. Abre el archivo `index.html` en tu navegador para ver el blog.
